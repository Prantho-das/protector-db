## Installation

protect-db requires [PHP](https://php.net/) v7+ to run.

Install the dependencies and devDependencies and start the server.

```sh
composer require pranthokumar/protect-db
```

## How To Work
``
route---- /protect-db
comm ---- php artisan protect-db:protect
config---- weakly setup
``
